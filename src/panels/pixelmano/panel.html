Az Élő Vászon

Ez a panel a Pulzus Műhely design-rendszere, ami önmagát dokumentálja. Minden komponens megmutatja a saját HTML-jét, CSS-ét és a felhasznált design-tokeneket. A téma-csúszkával valós időben átszínezed az egészet, és a rendszer azonnal megméri a WCAG AA kontrasztot. A dizájn itt nem statikus dísz, hanem élő, ellenőrizhető anyag: ha rossz színt állítasz be, a nagy cím a szemed előtt olvad bele a papírba, ahogy javítod, élesre pattan.

Téma-hangoló

8°
46%
A cím kontrasztja (nyers szín × papír) — AA
A gomb felirata (védett komponens) — AA

A nagy címhez (large text) a WCAG AA 3:1-et, a normál felirathoz 4,5:1-et kér. A cím a nyers akcent-szín: ha az beleolvad a papírba, eltűnik. A gomb ezzel szemben okosan ink- vagy papír-feliratot vált, hogy megvédje az olvashatóságát — ezt méri a második sor.

01

Gomb .ev-btn

Elsődleges és "ghost" variáns. A felirat színét a rendszer az akcenthez választja (ink vagy papír), hogy AA maradjon. Fókusz-gyűrű billentyűzethez.

  • --ev-accent
  • --ev-btn-fg
  • --ev-ink
  • --ev-line
<button class="ev-btn">Pulzus indítása</button>

.ev-btn {
  background: var(--ev-accent);
  color: var(--ev-btn-fg);   /* JS a jobb kontrasztút választja */
  border: 2px solid var(--ev-ink);
  padding: 0.55rem 1.1rem;
  text-transform: uppercase;
  letter-spacing: 0.04em;
}
.ev-btn:hover { transform: translateY(-2px); }

Papír-felület precíz hajszálvonal-kerettel és erős tipográfiai hierarchiával. A mélységet a ritmus és a kontraszt adja, nem a lágy árnyék.

  • --ev-paper
  • --ev-line
  • --ev-accent
  • --ev-ink-soft
<div class="ev-card-demo">...</div>

.ev-card-demo {
  background: var(--ev-paper);
  border: 1px solid var(--ev-line);
  border-top: 4px solid var(--ev-accent);
  padding: 1.1rem;
}

Jelvény & toast .ev-badge

Állapotjelző tag-ek és egy diszkrét toast, editorial ritmusban. A jelentést a forma és a felirat hordozza, nem csak a szín — így színvakon is olvasható.

  • --ev-line
  • --ev-accent
  • --ev-paper-2
<span class="ev-badge ev-badge--ok">Élő</span>

.ev-badge {
  border: 1.5px solid var(--ev-ink);
  padding: 2px 9px;
  text-transform: uppercase;
  letter-spacing: 0.06em;
  font-size: 0.7rem;
}
.ev-badge--ok::before { content: "●"; color: var(--ev-accent); }
