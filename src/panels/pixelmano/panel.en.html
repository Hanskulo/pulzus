The Living Canvas

This panel is the design system of Pulzus Műhely, and it documents itself. Every component shows its own HTML, CSS and the design tokens it uses. With the theme slider you recolor the whole thing in real time, and the system instantly measures the WCAG AA contrast. Design here isn't static decoration but living, checkable material: set a bad color and the big title melts into the paper before your eyes; fix it and it snaps back sharp.

For the big title (large text) WCAG AA asks 3:1, for normal labels 4.5:1. The title is the raw accent color: if it melts into the paper, it vanishes. The button, by contrast, smartly switches to ink or paper text to protect its readability — that's what the second row measures.

01

Button .ev-btn

Primary and "ghost" variant. The system picks the label color against the accent (ink or paper) so it stays AA. Keyboard focus ring included.

  • --ev-accent
  • --ev-btn-fg
  • --ev-ink
  • --ev-line
<button class="ev-btn">Start pulse</button>

.ev-btn {
  background: var(--ev-accent);
  color: var(--ev-btn-fg);   /* JS picks the higher-contrast one */
  border: 2px solid var(--ev-ink);
  padding: 0.55rem 1.1rem;
  text-transform: uppercase;
  letter-spacing: 0.04em;
}
.ev-btn:hover { transform: translateY(-2px); }
