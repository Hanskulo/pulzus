Die lebende Leinwand

Dieses Panel ist das Designsystem von Pulzus Műhely, und es dokumentiert sich selbst. Jede Komponente zeigt ihr eigenes HTML, CSS und die verwendeten Design-Tokens. Mit dem Theme-Regler färbst du das Ganze in Echtzeit um, und das System misst sofort den WCAG AA-Kontrast. Design ist hier keine statische Dekoration, sondern lebendiges, prüfbares Material: stell eine schlechte Farbe ein, und der große Titel verschmilzt vor deinen Augen mit dem Papier; korrigier sie, und er schnappt gestochen scharf zurück.

Theme-Regler

8°
46%
Titel-Kontrast (rohe Farbe × Papier) — AA
Button-Beschriftung (geschützte Komponente) — AA

Für den großen Titel (large text) verlangt WCAG AA 3:1, für normale Beschriftung 4,5:1. Der Titel ist die rohe Akzentfarbe: verschmilzt sie mit dem Papier, verschwindet sie. Der Button dagegen wechselt klug zu Tinten- oder Papierschrift, um seine Lesbarkeit zu schützen — das misst die zweite Zeile.

01

Button .ev-btn

Primär- und "Ghost"-Variante. Das System wählt die Schriftfarbe gegen den Akzent (Tinte oder Papier), damit AA erhalten bleibt. Fokusring für die Tastatur inklusive.

  • --ev-accent
  • --ev-btn-fg
  • --ev-ink
  • --ev-line
<button class="ev-btn">Puls starten</button>

.ev-btn {
  background: var(--ev-accent);
  color: var(--ev-btn-fg);   /* JS wählt den kontrastreicheren */
  border: 2px solid var(--ev-ink);
  padding: 0.55rem 1.1rem;
  text-transform: uppercase;
  letter-spacing: 0.04em;
}
.ev-btn:hover { transform: translateY(-2px); }
